# Router SOP（就地版）

本文件给 `lightdash-insight-router/SKILL.md` 配套使用，避免跨目录长路径依赖。

## 最小信息集（缺一先问）

- 项目：`projectUuid` 或唯一项目名
- 时间范围：最近 12 个月 / 指定年月 / 指定区间
- 指标目标：价格指数 / 销量 / 销售额

## 缺参追问顺序

1. 项目
2. 时间
3. 指标

默认（仅用户允许“先查”）：

- 时间：最近 12 个月
- 指标：fisher
- limit：100

## 类目默认与降级

- 用户未指定层级：优先 `cls_4`
- 若 `get_explore` 不存在 `*_cls_4`：用最细可用 `cls_N`（`cls_3 -> cls_2 -> cls_1`）
- 先做小枚举（单维度，limit 30~50）再加主筛选
- 未命中只允许降一级再枚举一次
- 仍未命中：返回 5~10 个候选值让用户点选

## 调用次数与停止条件

- 普通问题最多 3 次工具调用
- 类目校验/降级最多 4 次（多的 1 次只用于降级枚举）
- 连续失败 2 次：停止试错，输出
  - 已确认信息
  - 失败原因
  - 需补充项
  - 下一步建议（<=2 条）

## 输出格式

1. 结论
2. 关键数字
3. 口径说明
4. 下一步选项
